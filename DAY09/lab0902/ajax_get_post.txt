// 응답 요청을 처리하는 부분(req)
요청을 전송해서 통신 시작하기
- req.open(method, url, async, user, pwd)
- method : HTTp 메소드 GET, POST, PUT, DELETE
- url : 접근할 url
- async : 비동기 통신여부 (선택, 기본적으로는 비동기 true)
- user : 인증할 때의 사용자 이름(선택)
- pwd : 인증할 때의 암호(선택)

GET 방식 오픈의 경우 직접 send 메소드로 요청을 보내 서버와 통신을 시작
- req.open("GET", "data.txt");
- req.send(null);

POST 방식의 경우
- req.setRequestHeader("content-type", "application/x-www-form-urlencoded");
- req.send("name=" + encodeURIComponent(value));
